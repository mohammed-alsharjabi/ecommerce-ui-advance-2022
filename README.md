# Advanced E-commerce UI — Flutter Practice Project (2022)

A historical Flutter project created to practice e-commerce interfaces, reusable widgets, payment SDK exploration, and early state-management approaches.

> **Project type:** UI and technology-learning project  
> **Status:** Archived learning project from 2022

## العربية

مشروع تدريبي قديم من عام 2022 ركزت فيه على بناء واجهات متجر إلكتروني وتجربة مكتبات الدفع والتنقل وإدارة الحالة. لا أقدمه كمنتج تجاري مكتمل، بل كجزء من تطوري المبكر في Flutter.

## Practiced Areas

- Product browsing interfaces
- Cart and checkout UI
- Ratings and carousel components
- Stripe and MyFatoorah SDK exploration
- Local persistence
- HTTP requests
- GetX and Provider experimentation
- Reusable Flutter widgets

## Tech Stack

- Flutter and Dart
- GetX
- Provider
- HTTP
- SharedPreferences
- Stripe
- MyFatoorah
- Flutter SVG
- Carousel Slider

## Historical Note

This repository uses pre-Dart 3 dependencies and patterns that do not represent my current engineering standards. It remains public to show progression over time. Newer projects on this profile use stronger feature separation, BLoC/Cubit, dependency injection, and cleaner repository boundaries.

## Running Locally

Because this is an older project, dependency upgrades may be required before it builds on a current Flutter SDK.

```bash
flutter pub get
flutter run
```
